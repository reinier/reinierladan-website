# Development instructions

## Needed on your computer:
- Node
- Ruby
- Sass

## Run when starting:
- `npm install`

## Run when developing:
- `gulp`
- Local test server is now available at [localhost:8080](http://localhost:8080)

## Run to deploy to gh-pages branch:
- `gulp publish`
